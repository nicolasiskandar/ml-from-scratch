# Logistic Regression

Implementation of L2-regularized logistic regression implementation that
normalizes inputs, maintains cost history, and predicts classes by thresholding
sigmoid outputs.

## Results Video Demo

[![Logistic Regression Demo](https://img.youtube.com/vi/rCykDD_pxYY/0.jpg)](https://www.youtube.com/watch?v=rCykDD_pxYY)

## Key files
- `logistic_regression.py`: the core `LogisticRegression` class with `fit`,
  `predict`, and private helper methods.
- `test_model.ipynb`: notebook that shows sample data generation, training, and
  visual diagnostics for the probability contours.
