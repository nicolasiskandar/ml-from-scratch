# Linear Regression

Linear regression implementation that normalizes features,
computes gradients analytically, and tracks cost history as the loop converges.

## Results Video Demo

[![Linear Regression Demo](https://img.youtube.com/vi/9pGXD0r46cg/0.jpg)](https://www.youtube.com/watch?v=9pGXD0r46cg)

## Key files
- `linear_regression.py`: only dependency is `numpy`; exposes `LinearRegression`
  with `fit`, `predict`, and cost tracking.
- `test_model.ipynb`, `test_model_2.ipynb`: notebooks to test
  datasets, visualize loss, and compare values.
