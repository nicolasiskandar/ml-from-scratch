# K-Means Clustering

`KMeansClustering` implementation that can either accept a
fixed `k` or run elbow heuristic to choose
`k` automatically.

## Results Video Demo

[![K-Means Clustering Demo](https://img.youtube.com/vi/J-Ner5j5MtI/0.jpg)](https://www.youtube.com/watch?v=J-Ner5j5MtI)

## Files
- `k_means_clustering.py`: implements centroid initialization, assignment,
  reassignment, tolerance checking, and inertia tracking for the elbow method.
- `test_model.ipynb`, `test_model_2.ipynb`, `test_model_3.ipynb`: notebooks that
  test and visualize clusters on datasets.
